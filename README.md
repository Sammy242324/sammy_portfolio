# IT Portfolio

A modern static portfolio starter for an IT profile.

## What is included

- Hero section with profile summary
- About, skills, projects, education, and contact sections
- Responsive layout with polished styling
- Lightweight JavaScript for dynamic year display and reveal animation

## How to use

1. Open `index.html` in a browser, or serve the folder with a local static server.
2. Replace all placeholder text with your own profile details.
3. Update the project cards, certifications, and contact information.
